About These Predictions

Performance Equivalencies: Based on competitive middle-distance running data and proven conversion formulas

400m Formula: 400m time ≈ (800m time - 11) / 2 - Based on the "double and add 10-12 seconds" rule used in track

Mile Formula: Mile time ≈ 800m time × 2.2 - Reflects actual performance relationships in competitive track

Example: A 1:40 800m runner can typically run 0:45 for 400m and 3:40-3:45 for the mile (confirmed by elite athlete data like David Rudisha: 1:42 800m / 0:45.5 400m)

Note: 800m is a unique event requiring both speed and endurance. These predictions assume balanced training across all distances.
